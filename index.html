О проекте

Мы разрабатываем современное веб-приложение "Каталог фильмов", которое позволит пользователям:

  • Просматривать популярные фильмы и сериалы
  • Искать фильмы по названию, жанру или году выпуска
  • Создавать персонализированные коллекции
  • Добавлять фильмы в "Избранное" и "Хочу посмотреть"
  • Оставлять рецензии и оценки

Почему это важно? Приложение демонстрирует полный стек современных веб-технологий и представляет собой типичный пример реального коммерческого проекта.

Технологический стек

Проект будет реализован с использованием следующих технологий:

React + TypeScript

Типизированные функциональные компоненты с хуками для построения интерфейса

Redux Toolkit

Централизованное управление состоянием приложения с поддержкой асинхронных операций

React Router

Клиентская маршрутизация между 4+ страницами приложения

REST API

Интеграция с бэкендом для CRUD операций (Firebase/Supabase или собственный сервер)

Архитектура и ключевые функции

Главная страница

Слайдер популярных фильмов, рекомендации, новинки кино

Поиск и фильтрация

Расширенный поиск по жанрам, году, рейтингу и актерам

Детальная страница

Полная информация о фильме: трейлеры, актеры, рецензии, похожие фильмы

Личный кабинет

Управление коллекциями, история просмотров, рекомендации

Структура состояния (Redux)

  • movies - список фильмов с пагинацией
  • favorites - избранные фильмы пользователя
  • watchlist - список "хочу посмотреть"
  • user - данные авторизованного пользователя
  • reviews - рецензии и оценки

Стратегия тестирования

Unit-тесты

Jest + React Testing Library для тестирования редьюсеров, селекторов и хуков

Скриншотные тесты

Storybook + Loki для визуального регрессионного тестирования компонентов

E2E тесты

Cypress для тестирования ключевых пользовательских сценариев

Полное покрытие: Тесты будут охватывать 70%+ кодовой базы, включая критические пользовательские сценарии.

Расширенные возможности

Для получения высокой оценки планируется реализовать:

  • GraphQL API вместо REST для более эффективной загрузки данных
  • Интеграция с GPT для автоматической генерации описаний фильмов и рекомендаций
  • PWA (Progressive Web App) для оффлайн-доступа к избранным фильмам
  • WebSocket для реального времени уведомлений о новинках
  • Телеграм-бот для получения уведомлений и поиска фильмов

Инновации: Приложение будет не просто учебным проектом, а демонстрацией современных подходов к веб-разработке.
