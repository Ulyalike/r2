О проекте

Cinemate — это современная веб-платформа для кинолюбителей, которая выходит за рамки простого каталога фильмов. Мы создаем интеллектуальную экосистему, где пользователи не только открывают для себя новые фильмы, но и получают глубокий аналитический контент, персонализированные рекомендации и возможность совместного обсуждения кино.

В отличие от существующих решений, Cinemate интегрирует AI-анализ для автоматической категоризации, суммаризации рецензий и создания уникальных подборок на основе психологических профилей пользователей.

Наш проект реализуется в рамках учебного курса с применением современных технологий: TypeScript, React, Redux Toolkit и других инструментов профессиональной веб-разработки.

Ключевые возможности

AI-рекомендации

Персонализированные подборки фильмов на основе анализа предпочтений и поведения пользователя с использованием машинного обучения.

Коллективные обсуждения

Режим совместного просмотра и обсуждения фильмов в реальном времени с использованием WebRTC технологии.

Аналитика контента

Автоматическая суммаризация рецензий, анализ эмоциональной окраски фильмов и создание интерактивных графиков.

Кроссплатформенность

Полностью адаптивный интерфейс, работающий на всех устройствах от смартфонов до десктопных компьютеров.

Технологический стек

TypeScript
React
Redux Toolkit
React Router
Firebase/Supabase
Jest & Testing Library
Cypress
Storybook

Frontend

Современный React с TypeScript для типобезопасности, Redux Toolkit для управления состоянием, React Router для навигации между 4+ страницами.

Backend & API

Firebase/Supabase с REST API для хранения данных о фильмах, пользователях и рецензиях. Полный CRUD функционал (GET, POST, PUT, DELETE).

Тестирование

Полное покрытие тестами: unit-тесты (Jest), скриншотные тесты (Storybook), end-to-end тесты (Cypress) для ключевых сценариев.

Уникальность проекта

1

AI-интеграция для анализа контента

Использование локального сервера Ollama или API сторонних сервисов для автоматической суммаризации рецензий, определения эмоционального тона фильмов и создания умных категорий на основе контента, а не только метаданных.

2

WebRTC для совместного просмотра

Реализация возможности синхронного просмотра фильмов с друзьями с голосовым/текстовым чатом и синхронизацией времени воспроизведения без единого бэкенд-сервера.

3

Расширение для браузера

WebExtension для сохранения понравившихся фильмов из любых сайтов напрямую в Cinemate, анализа истории просмотров и получения рекомендаций на основе посещенных страниц.

4

Коммерческая перспектива

Платформа имеет четкую монетизацию: премиум-подписка для расширенной аналитики, партнерство со стриминговыми сервисами, продажа данных анонимизированной аналитики для кинокомпаний.
